# 🎓 South Indian Colleges – Data Analysis Project

A comprehensive data analysis project examining **500+ colleges across South India**, covering fees, courses, placements, and rankings. The project combines **Python** for data cleaning and analysis, **Power BI** for interactive dashboards, and **Excel** for detailed reporting.

---

## 📌 Project Overview

This project analyzes college-level data from South India to uncover meaningful trends and insights — helping students, parents, and researchers make informed decisions about higher education. The workflow spans the full data analysis pipeline: cleaning raw data in Python, performing exploratory analysis, building an Excel report, and visualizing key findings through an interactive Power BI dashboard.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| **Python** (Pandas, NumPy, Matplotlib/Seaborn) | Data cleaning, preprocessing, and exploratory analysis |
| **Microsoft Excel** | Structured reporting, pivot tables, and summary statistics |
| **Power BI** | Interactive dashboards and visual storytelling |

## 🚀 How to Use This Project

### 1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/south-indian-colleges-analysis.git
cd south-indian-colleges-analysis
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the analysis script
```bash
python scripts/data_analysis.py
```

### 4. Explore the outputs
- Open `excel-report/colleges_analysis.xlsx` for the detailed report
- Open `powerbi-dashboard/colleges_dashboard.pbix` in **Power BI Desktop** for interactive visuals
